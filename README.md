# Webservice - Pedidos

## Sobre o projeto !!!

Consiste em um webservice de pedidos, onde implementamos um conceito relacional. Criamos endpoint para Criar, Alterar, excluir e pesquisar usuários. As outras entitidas e relacionamentos serão inseridas manualmente via banco de dados.
